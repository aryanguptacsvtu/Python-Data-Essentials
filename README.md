# 📊 Python Data Essentials - NumPy & Pandas Learning Repository

A comprehensive collection of Jupyter notebooks demonstrating practical implementations and exercises in NumPy and Pandas for data analysis and manipulation.

## 📋 Overview

This repository contains hands-on Jupyter notebooks covering essential concepts and operations in two fundamental Python libraries for data science:
- **NumPy**: Numerical computing and array operations
- **Pandas**: Data manipulation and analysis

## 🚀 Getting Started

### Prerequisites

Ensure you have the following installed:
- Python 3.7 or higher
- Jupyter Notebook or JupyterLab
- NumPy
- Pandas

### Installation

1. Clone the repository:
```bash
git clone https://github.com/aryanguptacsvtu/Jupyter-Notebook.git
cd Jupyter-Notebook
```

2. Install required packages:
```bash
pip install -r requirements.txt
```

3. Launch Jupyter Notebook:
```bash
jupyter notebook
```

## 📚 Learning Modules

### NumPy Notebooks (Numpy1 - Numpy20)

The NumPy section covers:
- Array creation and initialization

- Array operations and broadcasting
- Mathematical and statistical functions
- Linear algebra operations
- Array manipulation and reshaping
- Boolean indexing and fancy indexing
- Random number generation
- File I/O operations

### Pandas Notebooks (Pandas1 - Pandas16)

The Pandas section covers:
- Series and DataFrame creation

- Data inspection and exploration
- Data selection and filtering
- Data cleaning and preprocessing
- Handling missing values
- Data aggregation and grouping
- Merging and joining datasets
- Data visualization basics
- Time series operations

3. Experiment with the code by modifying parameters
